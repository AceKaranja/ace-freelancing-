# Ace Freelancing Platform

A complete freelancing platform for Kenyan writers to earn money by completing academic writing tasks.

## Features

- User registration and authentication
- Account activation with M-Pesa STK Push
- Task management system
- Payment processing
- File submission system
- Withdrawal options
- Training program

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage for data persistence

## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in a web browser
3. The application will work immediately with no backend required

## Usage

1. **Registration**: Create an account with email and phone verification
2. **Activation**: Pay KES 450 activation fee via M-Pesa
3. **Browse Tasks**: View available writing tasks
4. **Select Tasks**: Choose one task per day to work on
5. **Submit Work**: Upload completed work in PDF, TXT, or JPEG format
6. **Get Paid**: Receive payments via M-Pesa, Airtel Money, Bank, or PayPal
